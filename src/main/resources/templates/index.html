Mais, la qualité de code, qu'est-ce que c'est?

La qualité de code est un ensemble de critères qui permettent de juger de la qualité d'un code source.

Il existe différentes méthodes permettant de rendre un code "propre"

Pour simplifier, il faut respecter les principes suivants :

1. Ne pas écrire de "franglais", les thermes doivent tous être en anglais

2. Ne pas utiliser de variables globales

3. Ne pas utiliser de variables de type "var"
